Integrations

Connect your trading account to your Guardian Layer. Choose the integration that matches your platform below.

Section 01

MetaTrader 4 / MetaTrader 5

Download the Guardian EA. Install it on any chart in MetaTrader. It monitors your account and reports trade activity to your Guardian Layer.

  • 01 Download the EA file below
  • 02 In MetaTrader: File → Open Data Folder → MQL4/Experts (or MQL5/Experts)
  • 03 Copy the .ex4 / .ex5 file into the Experts folder
  • 04 Restart MetaTrader, drag the EA onto any chart
  • 05 Paste your Guardian token when prompted (shown below)

Section 02

cTrader

Use the webhook integration. cTrader cBot can send trade events to EdgeKeeper via HTTPS.

  • 01 Configure your cBot to POST trade events to your Guardian webhook URL
  • 02 Set the Authorization header to: Bearer <your-token>
Guardian Webhook URL
https://edgekeeper.io/api/guardian/update?token=•••
Example Payload
{ "account_balance": 10250.00, "open_positions": 1, "daily_pnl": -120.00, "violation_type": "oversized_position" }
Section 03

TradingView

Use Pine Script alerts and webhook to send signals to your Guardian Layer.

  • 01 In your Pine Script alert, set the webhook URL to your Guardian endpoint
  • 02 Use the JSON format below in the Message field
Webhook URL
https://edgekeeper.io/api/guardian/update?token=•••
Alert Message (JSON)
{"account_balance": {{strategy.equity}}, "daily_pnl": {{strategy.netprofit}}}
